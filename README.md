# Список задач (Todo App)

Веб-приложение для просмотра и фильтрации задач с использованием React и Tailwind CSS.

## 🚀 Запуск проекта

### Требования

- Node.js (версия 18 или выше)
- npm или yarn

### Установка и запуск

```bash
# Перейдите в директорию проекта
cd todo

# Установите зависимости
npm install

# Запустите сервер разработки
npm run dev
```

Приложение будет доступно по адресу `http://localhost:5173`

### Сборка для продакшена

```bash
# Создать оптимизированную сборку
npm run build

# Предварительный просмотр сборки
npm run preview
```

## 🛠 Используемые технологии

- **React 18** - библиотека для создания пользовательского интерфейса
- **Vite** - быстрый инструмент сборки
- **Tailwind CSS 4** - утилитарный CSS-фреймворк
- **@tailwindcss/postcss** - PostCSS плагин для Tailwind CSS 4
- **JSONPlaceholder API** - публичный API для получения данных

## 📋 Функционал

### Основной экран
- Загрузка и отображение списка задач из API
- Показываются первые 15 задач
- Для каждой задачи отображается:
  - Название (`title`)
  - Статус выполнения (`completed`)
  - ID пользователя (`userId`)

### Фильтрация
- **Поиск** по названию задачи
- **Фильтр по статусу**:
  - Все задачи
  - Выполненные
  - Невыполненные

### Детальная карточка задачи
При клике на задачу открывается модальное окно с подробной информацией:
- Название задачи
- Статус выполнения
- ID задачи
- ID пользователя
- Информация о пользователе (из API users):
  - Имя (`name`)
  - Email (`email`)

### Обработка состояний
- **Loading** - индикатор загрузки при запросе к API
- **Empty** - сообщение, если по фильтру ничего не найдено
- **Error** - отображение ошибки с кнопкой повторной попытки

## 🔌 API

Приложение использует публичный API JSONPlaceholder:

- `https://jsonplaceholder.typicode.com/todos` - список задач
- `https://jsonplaceholder.typicode.com/users` - список пользователей
